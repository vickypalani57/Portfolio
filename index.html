My Project

EARTHQUAKE_PREDICTION_AI

This project aims to predict the potential locations of earthquakes using synthetic seismic data generated to simulate real-world earthquake patterns. By applying machine learning techniques in Python, the model learns from features like magnitude, depth, region type, and timestamp to estimate the most likely geographic zones where seismic activity may occur.

GROCERY_WEBSITE

This project is a responsive grocery website built by our team to provide users with a smooth and intuitive online grocery shopping experience. The platform allows users to browse products, add items to a cart, view offers, and simulate the checkout process.
